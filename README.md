# Agent Awake — Releases

This repository hosts the **downloadable builds** and the **Sparkle auto-update
feed** (`appcast.xml`) for **Agent Awake**.

- Get the app from the [Agent Awake website](https://fil.technology) (or the
  [Releases](../../releases) page here).
- Once installed, Agent Awake reads `appcast.xml` and updates itself
  automatically — you don't need anything from this repo to use the app.

Builds are published automatically by CI from the (private) source repository.
